What is semantic HTML?

Semantic HTML clearly describes its meaning to both the browser and the developer.

How do you import a stylesheet in to a HTML document?

Include a reference to the external style sheet file inside the element, inside the head section.
Example: <link rel="stylesheet" href="styles/subpage.css" type="text/css" />

How do you import javascript in to a HTML document

Use script tag at the end of the end html file
Example: <script src="myscripts.js">

What meta tag can you use to fix the mobile screen width?

'Viewpoint' meta tag.

How do you apply CSS only to certain screen sizes?

That can be achieved by using Media Queries

What does mobile first mean when writing CSS?

A mobile-first approach means that styles are applied first to mobile devices. Advanced styles and other overrides for larger screens are then added into the stylesheet via media queries.

How does a browser decide the order to apply CSS?

A browser decides the order in a way that the most specific selector gets the highest precedence
